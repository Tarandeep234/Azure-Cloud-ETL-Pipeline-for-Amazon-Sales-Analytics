Azure Amazon Sales ETL Pipeline
What This Project Does

This project demonstrates a simple ETL pipeline built in Azure Data Factory.

The pipeline reads Amazon sales data from a GitHub CSV file (HTTP source) and loads it into Azure Data Lake Storage Gen2.
